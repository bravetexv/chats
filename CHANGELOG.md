# Multi-Platform Chat Manager

## Versión 0.0.0 - Changelog

### Correcciones
- ✅ Solucionado problema de duplicación de mensajes en Twitch y Kick
- ✅ Mejorada estabilidad de conexión con timeout de 10 segundos
- ✅ Optimizado manejo de event listeners para prevenir duplicados

### Nuevas Funcionalidades
- ⚡ TTS: Cancelación inmediata al desactivar (no espera mensajes pendientes)
- ⚡ Cambio de canal directo sin necesidad de desconectar
- ⚡ Guardado automático del canal conectado

### Archivos Ejecutables

**Portable** (Recomendado):
- `Multi-Platform Chat Manager 0.0.0.exe` - No requiere instalación

**Instalador**:
- `Multi-Platform Chat Manager Setup 0.0.0.exe` - Instalación en el sistema

### Uso Rápido

1. Ejecuta el archivo `.exe`
2. Configura tus conexiones en ⚙️ Configuración
3. Conecta Twitch, YouTube o Kick

### Notas Técnicas
- Mantiene últimos 500 mensajes en memoria
- Logs detallados para debugging
- Soporte para múltiples plataformas simultáneas
